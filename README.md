# CUDA & OpenMP on GPU — small university comparison project

Small university project exploring GPU programming with **CUDA** and **OpenMP offloading** on an NVIDIA GPU, with a focus on **runtime/performance** and **developer experience**.

This repository contains two main parts:
- **Matrix multiplication benchmarks** (`MatrixMult/`) — CUDA implementations plus cuBLAS variants
- **Parallel Cyclic Reduction (PCR) solver** (`PCR/c/`) — implementations in plain C/OpenMP, CUDA, and OpenMP-offload-to-GPU

> Note: This code is kept “as is” (no further updates).
